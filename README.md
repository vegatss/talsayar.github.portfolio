# dark-founder-portfolio

A minimal dark one-page portfolio starter for solo builders, founders, and product engineers.

## Includes

- responsive single-page layout
- dark premium palette
- hero, direction, focus, and stack sections
- no build step required

## Usage

Open `index.html` and replace the placeholder text, links, and colors with your own brand.
